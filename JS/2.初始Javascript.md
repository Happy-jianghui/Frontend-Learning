## 1.2 Javascript是什么
1. JavaScript是世界上最流行的语言之一，是一种运行在客户端的脚本语言（Script是脚本的意思）
2. 脚本语言：不需要编译，运行过程中由js解释器（js引擎）逐行来进行解释并执行
3. 现在也可以基于Node.js技术进行服务器端编程
4. JavaScript简称JS
