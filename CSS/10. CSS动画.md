# 1. 2D 转换
**转换（transform）** 是CSS3中具有颠覆性的特性之一，可以实现元素的位移、旋转、缩放等效果。  
 - 移动：translate
 - 旋转：rotate
 - 缩放：scale

## 1.1 二维坐标系
2D转换是改变标签在二维平面上的位置和形状的一种技术
